    Here are some examples that can help you to input address correctly.

    1. It is not necessary to input nor the country name nor the definition of administrative division (like "city", "village", "district") or "state" (if USA) to get the Google photo of a particular object according to the address entered. But this may be a necessary refinement if you can't get right result.

      Street: [Street name], [the number object] City: [City name]

      Street: [the number object], [Street name] City: [City name]

      Street: 73, Franklin Avenue City: Brooklin

      Street: Галицька, 7 City: Червоноград

    2. If you entered the address and the result seemes strange (the objects shown cannot belong to the area), verify that you entered the address correctly. Also result can be unexpectable and wrong if You fill only one field with address.

    3. It is better to follow the rules of transliteration, if you input the address of object that belongs to a non-English speaking country.

      You can use transliteration online there.

      Sometimes it is possible to get a right result if the address will be entered in different languages in each field (if the languages are somewhere similar).

      Street: 25 Konstantine Gamsakhurdia St City: ბათუმი

    4. Трапляються випадки, коли в полі "Вулиця" потрібно вводити номер об'єкту (будівлі) у формі діапазону [номер-номер], щоб отримати фотографію об'єкту.

      Street: Peter-Rosegger-Straße, 42-44 City: Graz

    5. If the object is fairly famous (this is a monument of architecture, a museum, a stadium, a park, or any place of interest), You can input the address as follows. Unfortunately, some famous places are still not visible.

      Street: Castel dell'Ovo City: Napoli

      Street: Goblin Valley State Park City: Utah

      Street: Пляж Ланжерон City: Одеса

    6. Some places can not be found on the map: low-populated or unpolluted objects, objects related to cultural or natural heritage, little-known villages and settlements, etc.
